## 这里写接口名称

#### 请求地址

<u class="post">POST</u> `/api/auth/login/{user}`

#### URL 参数

| 字段 | 必填 | 类型 | 详细描述 |
| ---- | ---- | ---- | -------- |
| user | 是 | INT | 用户 UID |

#### 请求参数

| 字段 | 必填 | 类型 | 详细描述 |
| ---- | ---- | ---- | -------- |
| code | 是 | STRING | 微信小程序授权码 |

#### 响应参数

| 字段 | 类型 | 详细描述 |
| ---- | ---- | -------- |
| token | STRING | 登录凭证 |

> 一段重要的内容，可以和其他 **Markdown** 语法混用。

!> 一段重要的内容，可以和其他 **Markdown** 语法混用。

?> 一段重要的内容，可以和其他 **Markdown** 语法混用。

#### 响应成功样例

```json
{
    "code": 0,
    "message": "success",
    "data": {
        "token": "bb6e6721562d0ad6468d4c57a4fb19cae04541e2"
    }
}
```
